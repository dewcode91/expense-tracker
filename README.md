# Expense Tracker (Android)

Kotlin + Jetpack Compose + Room expense tracker starter.

## Features scaffolded
- Dashboard summary
- Categories
- Recurring expenses
- Budgets
- Charts (placeholder)
- CSV export to Downloads
- Receipt photo picker

## Next steps
- Add real data entry forms
- Add charts with a charting library
- Hook receipts to expenses
- Add filters (date range, category)

## Run
Open in Android Studio and run the `app` module.
